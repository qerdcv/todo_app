# TODO APP IN GO


## Practice project

### Dependencies

* Docker
* docker-compose
* Make


### Quick start

To run with local config
```shell
make run-local
```
To attach to the logs of the containers
```shell
make logs-local
```
To clean-up
```shell
make rm-local
```

Same for production but
```shell
make run-prod
make logs-prod
make rm-prod
```
